# Website Redesign Project Brief: QS Global Solutions

## Project Overview
Redesign the existing Quantity Surveying Global Solutions website (qs-global-solutions.com) with a modern, professional aesthetic while maintaining all existing content and SEO metadata.

## Critical Requirements

### SEO Preservation
- Retain all existing SEO meta titles, descriptions, and keywords exactly as they are
- Maintain current URL structure and permalink format
- Preserve all heading hierarchy (H1, H2, H3) for SEO purposes
- Keep all alt text for images unchanged

### Content Retention
- Use all existing written content from each page without modification
- Maintain the same information architecture and page structure
- Preserve all contact information, email addresses, and phone numbers
- Keep all service descriptions, features, and benefits as written

## Design Direction

### Modern Design Elements
- Clean, contemporary layout with generous white space
- Smooth page transitions and scroll animations
- Hover effects on buttons, cards, and interactive elements
- Subtle parallax scrolling effects on hero sections
- Fade-in animations for content sections as users scroll
- Modern typography with clear hierarchy (consider using Google Fonts like Inter, Poppins, or Montserrat)
- Professional color palette (suggest: deep navy/blue as primary, complemented by white, light gray, and accent colors)

### Visual Effects & Interactions
- Animated number counters for statistics/achievements
- Smooth slide-in animations for service cards
- Glassmorphism or soft shadows for elevated elements
- Gradient overlays on hero images
- Button micro-interactions (scale, color shift on hover)
- Loading animations between page sections
- Animated iconography for services and features
- Progress indicators for multi-step forms

### Layout Enhancements
- Full-width hero sections with compelling imagery
- Card-based layouts for services with hover effects
- Grid systems for displaying project types and expertise areas
- Sticky navigation bar that changes on scroll
- Mobile-first responsive design
- Split-screen layouts for key comparison sections
- Icon-driven feature presentations
- Testimonial carousels with smooth transitions

### Images & Media
- Use high-quality placeholder images related to:
  - Construction sites and buildings
  - Architectural drawings and blueprints
  - Professional business settings
  - Modern office environments
  - Global landmarks (Gulf region, UK, Australia)
  - Team collaboration scenes
  - Technology and software interfaces
- All images are temporary placeholders (will be replaced with actual project images later)

## Technical Specifications

### Performance
- Optimize all animations for smooth 60fps performance
- Lazy loading for images and heavy content
- Minimize CSS and JavaScript for fast load times
- Ensure mobile performance optimization

### Responsive Design
- Fully responsive across desktop (1920px), tablet (768px), and mobile (375px) breakpoints
- Touch-friendly interface elements for mobile
- Appropriate font scaling across devices
- Stackable layouts for mobile views

### Accessibility
- Maintain WCAG 2.1 AA compliance
- Proper contrast ratios for text and backgrounds
- Keyboard navigation support
- Screen reader compatibility
- Focus states for interactive elements

### Framework/Technology Suggestions
- Modern CSS (Flexbox, Grid, CSS animations)
- Consider using: Tailwind CSS, Bootstrap 5, or custom CSS
- JavaScript/jQuery for interactions (or modern frameworks like React, Vue)
- Smooth scrolling libraries (e.g., GSAP, AOS, ScrollReveal)
